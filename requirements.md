# Goal

Create a simple Progressive web app hosted on github pages that helps me track my
workout progress during my workout. 


# Requirements

- static webpage (or at least no server costs)
- no permanent data storage
- An exercise "bank" can be read in from a .csv file or similar so that I can 
modify the exercises and then push the updates to github. 
- I want to be able to select from the exercise exercise bank to start a new workout
- and then as I check off exercises, they disappear. 
- I want to be able to group the selected exercises together (i.e. a superset)
- I want to have a separate 'stretch bank' with the same functionality
- I dont need pictures. I dont need you to create any content really
- I want to be able to open the PWA from my phone (e.g., safari or chome add to home screen)
- It needs to be able to keep the "session" active for up to 2 hours - so if it refreshes 
within taht time period (or reloads automatically or something - idk what ios might do) i dont
want to lose my progress. 
- I dont need to store what I've done or the weights or anything like that. 

- I have been using reminders for this on my iphone, which is kind of what I want, but
reminders once you check something off its gone forever and its hard to develop an exercise bank
- The UI should be minimalist with a basic dark/light mode.
